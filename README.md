# Bamba AI Website

このリポジトリは、個人事業 **Bamba AI** のホームページです。

ホームページURL: https://bamba-ai.com

## Bamba AIについて

Bamba AIは、東京都品川区周辺の個人店・中小企業向けに、AI活用支援、SNS活用支援、PR動画作成・配信、Webサービス開発を行う個人事業です。

「地元の商売に、ちょうどいいAI活用支援」をコンセプトに、難しいIT導入ではなく、日々の集客や情報発信にすぐ使える実践的な支援を提供します。

## 主な支援内容

- SNS投稿の企画・文章作成
- 店舗やサービスのPR動画制作
- Googleビジネスプロフィールや各種SNSでの発信支援
- ChatGPTなどのAIツールを活用した業務効率化
- Webサービス開発

地域のお店や事業者の魅力をわかりやすく伝えるお手伝いをします。

## 画像素材

`images` フォルダ以下には、Bamba AIのホームページで利用する画像素材を配置します。

- Bamba AIのロゴ
- ビジョンのイメージ
- LINEのQRコード

サイト内で必要に応じて適宜利用してください。

## 技術要件

- フロントエンド: Next.js（App Router）
- ホスティング: Cloudflare Pages
- ドメイン: bamba-ai.com（Cloudflareで取得済み）
- ホームページURL: https://bamba-ai.com
- コード管理: https://github.com/bamba-ai/www
- CI/CD: GitHubのソースコードを更新すると、自動的にCloudflare Pagesも更新される構成にする
